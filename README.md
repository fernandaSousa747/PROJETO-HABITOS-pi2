# Sistema de Controle de Hábitos

## Descrição
Sistema simples em Python para gerenciar hábitos saudáveis.

## Funcionalidades
- Cadastro de usuários
- Adição de hábitos
- Listagem de hábitos

## Tecnologias
- Python

## Possíveis usos da nossa solução
Ajuda pessoas a organizarem sua rotina e desenvolverem hábitos saudáveis.
