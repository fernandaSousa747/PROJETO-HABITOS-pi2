## Estrutura do VIDA+

O projeto está organizado de forma simples, contendo um arquivo principal responsável pela execução do sistema:

- `main.py`: arquivo principal onde estão implementadas as classes e a lógica do sistema.



## Explicação sobre as classes usadas no projeto

O sistema foi desenvolvido utilizando Programação Orientada a Objetos, com três classes principais:

1º Usuario:
Representa o usuário do sistema, contendo informações como nome, idade e uma lista de hábitos. Também possui métodos para adicionar e listar hábitos, permitindo o acompanhamento das atividades cadastradas.

2º Habito
Representa um hábito que o usuário deseja acompanhar, contendo atributos como nome, descrição e frequência. Essa classe organiza as informações relacionadas às atividades.

3º Sistema
Responsável por gerenciar os usuários do sistema, permitindo o cadastro e a organização das informações.

## Possíveis usos da nossa solução

O sistema de controle de hábitos pode ser utilizado por pessoas que desejam organizar melhor sua rotina e desenvolver comportamentos mais saudáveis no dia a dia, como estudar com frequência, praticar exercícios ou melhorar a disciplina pessoal.

Além do uso individual, a solução também pode ser aplicada em contextos educacionais ou sociais, como projetos que incentivam jovens a adotarem hábitos positivos. Nesse cenário, o sistema pode auxiliar no acompanhamento de atividades e no estímulo à constância, contribuindo para o desenvolvimento pessoal.

Em um contexto mais amplo, pequenas iniciativas ou profissionais da área de educação e saúde podem utilizar uma ferramenta semelhante para acompanhar a evolução de alunos ou clientes, promovendo maior organização e acompanhamento das atividades realizadas.

Esse projeto surge uma necessidade pessoal de organização e criação de hábitos mais saudáveis alinhado a uma rotina de inclui trabalho, estudo e ativdade fisica.
